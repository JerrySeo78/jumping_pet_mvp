# 🐾 Jumping Pet (점핑펫) - Prototype

Doodle Jump 스타일의 세로 스크롤 점프 캐주얼 게임 프로토타입.

## 🎮 플레이

- **PC**: 좌/우 방향키 또는 A/D키로 이동
- **모바일**: 터치 드래그로 이동
- 플랫폼을 밟으며 위로 올라가세요!

## 🧊 플랫폼 종류

| 타입 | 색상 | 설명 |
|------|------|------|
| Normal | 파란색 | 기본 플랫폼 |
| Break | 보라색 | 한번 밟으면 부서짐 |
| Moving | 파란색 | 좌우로 이동 |
| Spring | 초록색 | 높이 점프 |

## ⚡ 피처

- FEVER 게이지 시스템 (피버 시 배경 변환 + 점프력 증가)
- 캔디 수집
- 레벨 & 높이(m) 시스템
- 파티클 이펙트
- 스크린 쉐이크
- 하이스코어 로컬 저장

## 🚀 실행

별도 빌드 없이 `index.html`을 브라우저에서 열면 됩니다.

또는:
```bash
npx serve .
```

## 🛠 기술 스택

- Vanilla HTML5 Canvas
- Zero dependencies
- Mobile-responsive
